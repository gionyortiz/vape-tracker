## üõ†Ô∏è Troubleshooting

### Common Questions

**Q: Why don't I see the Customers page?**  
A: Click "Customers" in the navigation menu. If blank, check browser console (F12) for errors.

**Q: Customer not earning points?**  
A: Ensure customer is selected in sales page before checkout. Points are awarded automatically when customerId is present in transaction.

**Q: How do I import existing customers?**  
A: Use Data Manager page ‚Üí Import Data, or add manually via Customers ‚Üí Add Customer.

**Q: Can I customize loyalty tiers?**  
A: Yes! Edit `app.loyaltyManager.loyaltyConfig` in `js/customer-loyalty.js` (lines 11-30).

**Q: How do I connect email marketing?**  
A: Marketing messages are queued in localStorage. Integrate SendGrid/Mailchimp API in `queueMarketingMessage()` function (line 220).

**Q: Will this work on mobile?**  
A: Yes! Fully responsive design. Install as PWA for app-like experience.
